
# 2026 MCM Problem B: Colony Cost Optimization

本项目旨在建立数学模型，优化从2050年开始向火星（或其他目的地）运输1亿吨物资的成本与时间。

## 1. 优化模型设计
### 目标
*   **目标函数1**：最小化总成本 (Minimize Total Cost)
*   **目标函数2**：最小化运输时间 (Minimize Transport Time)

### 约束与参数
1.  **总运量**：1亿吨（100 million tons）。
2.  **运输方式**：
    *   **火箭 (Rockets)**：
        *   发射频次：最大 10次/天。
        *   单次运载量：150吨 (150,000 kg)。
        *   成本计算：动态预测（详见下文）。
    *   **太空电梯 (Space Elevators)**：
        *   数量：3台。
        *   容量：每台每年最多 179,000 吨。
        *   **太空电梯 (Space Elevators)**：
        *   数量：3台。
        *   容量：每台每年最多 179,000 吨。
        *   成本：TODO
3.  **预测模型**：火箭制造成本、燃料价格及太空电梯运营成本均随时间动态变化。

## 2. 火箭发射成本计算 (Rocket Launch Cost)
单次发射的总成本通过 `calculate_total_cost.py` 计算，公式如下：

$$ \text{Cost} = M_{load} \times \left[ \frac{C_{mfg}}{N} + R_{prop} \times (0.3 \times P_{fuel} + 0.7 \times P_{ox}) + M_{maint} \right] $$

**参数解释**：
*   **$C_{mfg}$ (Construction Cost)**: 火箭制造成本 ($/kg)。基于历史数据（含SpaceX数据）采用指数衰减模型预测，随技术成熟逐年降低。
*   **$N$ (Reuses)**: 火箭复用次数，默认为 20 次。这是降低成本的关键因素。
*   **$P_{fuel}$ (Fuel Price)**: 燃料价格 ($/kg)。基于历史数据通过统计模型预测。
*   **$R_{prop}$ (Propellant Ratio)**: 推进剂占火箭总重的比例，取 91%。
*   **$P_{ox}$**: 氧化剂等其他推进剂成本，视为常数 ($0.15/kg)。
*   **$M_{maint}$**: 维护成本，定为制造成本的 1%。

## 3. 模型实现与输出
优化模型 (`optimization_model.py`) 现采用 **NSGA-II (非支配排序遗传算法)** 进行多目标求解 (`pymoo` 库支持)。

### 算法特性
*   **决策变量**：日均火箭发射次数 (`launch_per_day`)，视为连续变量优化。
*   **多目标**：同时最小化时间 (Time) 和 成本 (Cost)。
*   **动态环境**：每一年的成本根据当年预测值动态计算。

### 输出结果
模型输出结果保存至 `optimization_results_nsga2.csv`，包含 Pareto 前沿上的所有最优解。控制台将推荐三种典型策略：
1.  **极速策略 (Minimize Time)**：高频发射，追求最短工期。
2.  **经济策略 (Minimize Cost)**：低频发射，最大化利用前期太空电梯的廉价运力。
3.  **平衡策略 (Balanced)**：在 Pareto 前沿的中位数取值，平衡资金与时间。

## 4. 火箭事故率预测 (Rocket Accident Probability)
基于 `Space_Corrected.csv` 的历史发射数据，并结合对未来商业航天成熟度的假设，建立了修正的贝叶斯预测模型 (`predict_rocket_accident.py`)。

### 建模方法
1.  **数据校准**：仅使用最近20年（2000-2020）的数据，因为现代数据更能反映当前的技术趋势。
2.  **强先验贝叶斯 (Strong Prior)**：
    *   引入了高可信度的先验假设（$\alpha=9990, \beta=10$），对应 0.1% 的基础失效率。这模拟了未来航天技术向民航安全标准靠拢的预期，修正了历史数据中包含大量早期实验性失败带来的偏差。
    *   修正后的2020年基准失效率为 **0.8115%**。
3.  **时间衰减 (Technological Decay)**：
    *   假设年均技术进步率为 **1%**（即失效率每年相对降低 1%），用于模拟 2050 年后的成熟期表现。

### 预测结果 (2050-2100)
该模型预测了自 2050 年起的火箭事故率如下：

| 年份 (Year) | 预测事故率 (Failure Rate) |
| :--- | :--- |
| **2050** | **0.6003%** |
| 2060 | 0.5429% |
| **2070** | **0.4910%** (跌破 0.5%) |
| 2080 | 0.4440% |
| 2090 | 0.4016% |
| **2100** | **0.3632%** |

此结果目前符合未来商业航天高可靠性（<1%）的预期，并在模型中作为风险与额外成本估算的依据。
